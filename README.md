# NZART Amateur Radio Certificate Study Guide

A study guide for the New Zealand Association of Radio Transmitters (NZART) Amateur Radio Certificate exam. This is a **work in progress**.

## Overview

A single-page HTML app covering all exam sections — from regulations and frequencies through to digital modes — with study notes and interactive practice questions for each topic.

## Sections

1. Regulations
2. Frequencies
3. Electronics Fundamentals
4. Measurement
5. Ohm's Law
6. Resistance
7. Power Calculations
8. Alternating Current
9. Capacitors, Inductors & Resonance
10. Safety
11. Semiconductors
12. Device Recognition
13. Meters and Measuring
14. Station Components
15. Receiver Block Diagrams
16. Receiver Fundamentals
17. Transmitter Block Diagrams
18. Harmonics and Parasitics
19. Power Supplies
20. Regulated Power Supplies
21. Modes and Emissions
22. Propagation

Each section includes a question bank with revealable answers.
